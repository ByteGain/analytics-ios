bg-analytics-ios is an iOS client for ByteGain based on Segment IO's library.

## Installation

BytegainAnalytics is available through [CocoaPods](http://cocoapods.org)

### CocoaPods

```ruby
pod "BytegainAnalytics", "4.0.1"
```

## Quickstart
